# Chat - AI
@Author : Varshith B R
AI Chat App using Generative Language Client API


## Setup Procedure

- Copy .env.example to .env
- Get API key from: [api-key](https://aistudio.google.com/app/apikey)
- `npm install`
- `npm run dev`
- `npm install react-icons`
- `npm install react-syntax-highlighter`
